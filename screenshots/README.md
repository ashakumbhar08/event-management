# Screenshots

Add project screenshots here with the following naming convention:

- `landing-page.png` - Landing page
- `user-dashboard.png` - User dashboard
- `browse-events.png` - Browse events page
- `admin-dashboard.png` - Admin dashboard
- `manage-events.png` - Event management page
- `analytics.png` - Analytics dashboard

## Recommended Screenshot Dimensions
- Width: 1920px (or 1280px minimum)
- Format: PNG or JPG
- Quality: High resolution for clarity
